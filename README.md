# KnightTravails

KnightTravails provides a recursive solution to find the minimum amount of moves a knight would need to make to travel the distance between 2 given squares.

## Usage

### Setup

1. Clone the repo
```bash
git clone https://github.com/JKanaiya/KnightTravailscd
```
2. Install dependencies
```bash
npm install
```

4. Give the coordinates
```bash
# Give the coordinates
node KnightMoves.js <coord1, coord2> <coord1, coord2>
```


Features
--------

- Displays each location the knight uses to get to the given location
- Gives the total moves the knight took to get there
